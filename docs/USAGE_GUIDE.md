## 1. はじめに

### EduShiftとは？

EduShiftは、中学受験・高校受験の国語記述問題を自動採点するAIシステムです。

**特徴：**
- ✅ 手書き答案をAIが読み取り
- ✅ 模範解答と照合して採点
- ✅ 良い点・改善点・書き直し例を提示
- ✅ 採点の厳しさを3段階で調整可能
- ✅ 複数生徒の一括採点に対応
- ✅ 採点結果をPDFレポートとして印刷・保存

### 採点の5大原則

EduShiftは以下の5つの観点で採点を行います：

| 観点 | 説明 |
|------|------|
| **1. 設問要求** | 「〜とはどういうことか」「なぜか」など設問の要求に合っているか |
| **2. 本文根拠** | 本文の内容に基づいた解答になっているか |
| **3. 指定条件** | 字数制限、キーワード、「本文中の言葉を使って」などの条件を満たしているか |
| **4. 減点ルール** | 誤字脱字、主述のねじれ、文末表現の誤りなど |
| **5. 文章形式** | 句読点、接続詞、文の構成が適切か |

## 2. 採点を始める

### Step 1: ファイルの準備

以下のファイルを用意してください：

| ファイル | 内容 | 例 |
|----------|------|-----|
| **生徒答案** | 生徒が書いた解答の写真・スキャン | 手書きノート、解答用紙の写真 |
| **模範解答** | 問題文と模範解答が載っている資料 | 問題集の解答ページ、塾のプリント |

> 💡 **ヒント**: 模範解答はテキストで直接入力することもできます。画像がない場合はテキスト入力モードをご利用ください。

### Step 2: アップロード

1. 「生徒の解答をアップロード」欄に**生徒答案**をドラッグ＆ドロップ（またはクリックしてファイルを選択）
2. 「解答・問題画像をアップロード」欄に**模範解答**をドラッグ＆ドロップ

> 💡 **スマホをお使いの場合**: アップロード欄をタップするとカメラが起動します。その場で答案を撮影し、そのままアップロードできます。撮影した画像は自動的に圧縮されるため、高画質のまま撮影して大丈夫です。

### Step 3: ファイルの役割を指定

ファイルをアップロードすると、各ファイルの役割を選ぶポップアップが表示されます。

| 役割 | 説明 |
|------|------|
| **答案** | 生徒が書いた解答 |
| **問題** | 問題文のみ |
| **模範解答** | 模範解答のみ |
| **問題＋模範解答** | 問題文と模範解答が同じファイルに含まれている場合 |

> 💡 **ヒント**: 問題集の解答ページなど、問題と模範解答が一緒に載っている場合は「問題＋模範解答」を選んでください。

**PDFの場合**: 複数ページのPDFでは、答案のページ・問題のページ・模範解答のページを個別に指定できます。

### Step 4: 問題番号の入力

採点したい問題番号を入力します。3つの入力形式から選べます：

| 形式 | 入力例 | 使い方 |
|------|--------|--------|
| **大問＋小問** | 大問1 問1、大問2 (1) | 大問番号と小問番号を組み合わせて指定 |
| **小問のみ** | 問1、(1)、(a) | 小問番号のみで指定 |
| **自由入力** | 任意の文字列 | 自由に問題を記述 |

> 💡 **ヒント**: 模範解答の問題番号と同じ形式で入力すると、AIが正確に問題を特定できます。

**複数問題の一括追加**: 開始番号と終了番号を指定すると、まとめて問題を追加できます。例えば「問1〜問5」を一度に追加できます。

### Step 5: 採点の厳しさを選択（任意）

| レベル | 説明 | おすすめの場面 |
|--------|------|----------------|
| **甘め** | 部分点を多めに付与、表現の揺れを許容 | 初めて記述に挑戦する生徒、モチベーション重視 |
| **標準** | 一般的な入試基準で採点 | 通常の演習、実力確認 |
| **厳しめ** | 細かな条件も厳格に判定 | 入試直前期、本番想定の演習 |

### Step 6: 採点実行

「採点する」ボタンをクリックして、結果を待ちます（通常10〜30秒）。

AIがまず手書き文字を読み取り、その後に採点を行います。読み取り結果の確認画面が表示されるので、正しく読み取れているか確認してください。

## 3. 採点結果の見方

### 採点レポートの構成

```
┌─────────────────────────────────────────┐
│  得点: 8/10点                            │
├─────────────────────────────────────────┤
│  📝 認識されたテキスト                    │
│  「主人公は〜と感じていた。」              │
├─────────────────────────────────────────┤
│  ✅ 良い点                               │
│  ・本文の表現を適切に引用している          │
│  ・設問の要求に沿った解答形式             │
├─────────────────────────────────────────┤
│  📈 改善アドバイス                        │
│  ・「なぜなら」の根拠がやや弱い            │
│  ・結論部分をより具体的に                  │
├─────────────────────────────────────────┤
│  ✏️ 書き直し例                           │
│  「主人公は〜という経験から、              │
│   〜と感じていたと考えられる。」           │
├─────────────────────────────────────────┤
│  📉 減点理由                             │
│  ・根拠の不足 (-2点)                      │
└─────────────────────────────────────────┘
```

### 各項目の活用法

| 項目 | 活用法 |
|------|--------|
| **得点** | 現在の実力把握、目標点との差を確認 |
| **認識テキスト** | AIが正しく読み取れているか確認（誤読があれば再採点で修正可能） |
| **良い点** | 生徒へのフィードバック、継続すべきポイント |
| **改善アドバイス** | 次回の記述で意識するポイント |
| **書き直し例** | 模範的な解答の書き方を学習 |
| **減点理由** | 具体的な失点原因を把握 |

> 💡 **ヒント**: 「良い点」「改善アドバイス」「書き直し例」は、必要に応じて手動で編集することもできます。

### PDFレポートの印刷・保存

採点結果はPDFレポートとして印刷・保存できます。生徒名や添削担当者名を入力しておくと、レポートに記載されます。

1. 採点結果画面で「PDFを印刷」ボタンをクリック
2. 印刷ダイアログが表示されます
3. プリンターで印刷するか、「PDFとして保存」を選択

> ⚠️ **ご注意**: 採点結果はページを閉じると保持されません。必要な結果は印刷またはPDF保存しておくことをおすすめします。

## 4. 再採点する

採点結果に納得がいかない場合や、読み取り結果に誤りがあった場合は、再採点ができます。

### 厳しさを変えて再採点

採点結果の画面に表示される以下のボタンから再採点できます：

| ボタン | 説明 |
|--------|------|
| **「もっと厳しくで再採点」** | より厳格な基準で採点し直します |
| **「もっと甘くで再採点」** | より寛容な基準で採点し直します |

### AI読み取り結果を修正して再採点

AIの文字読み取り結果に誤りがある場合は、認識テキストを手動で修正してから再採点することもできます。

1. 採点結果画面で「読み取り結果を修正」をクリック
2. テキストを正しい内容に修正
3. 「修正して再採点」をクリック

> 💡 **ヒント**: 字数制限などの問題条件がAIに正しく認識されなかった場合は、問題条件を手動で指定することもできます。

### 再採点のルール

- 初回採点後、**無料で2回まで**再採点できます
- 再採点では**採点回数は消費されません**（追加料金なし）
- 再採点は採点後**7日以内**に行ってください

> 💡 **ヒント**: まず「標準」で採点して結果を確認し、必要に応じて厳しさを変えて再採点するのがおすすめです。

## 5. 問題設定を保存する

同じ問題で繰り返し採点する場合、問題設定を保存しておくと便利です。

### 問題の保存方法

1. 採点画面で問題ファイル・模範解答・問題番号などを設定した状態で「問題を保存」をクリック
2. わかりやすい問題名をつけて保存（例：「2025年 開成中 大問2 問3」）

### 保存される内容

| 保存される | 保存されない |
|-----------|-------------|
| 問題ラベル（問題番号） | 生徒の答案ファイル |
| 配点・採点形式 | |
| 問題ファイル・模範解答ファイル | |
| 模範解答テキスト | |

### 保存した問題の読み込み

保存済みの問題一覧から選択すると、保存した設定が自動で読み込まれます。あとは生徒の答案をアップロードするだけで採点できます。

一覧では各問題のタイトル、問題数、ファイル数、合計サイズが表示されます。不要になった問題は一覧から削除することもできます。

### 保存に関する注意事項

> ⚠️ **ご注意**:
> - 保存先はお使いのブラウザ内です
> - 別のブラウザやデバイスからは利用できません
> - ブラウザのデータを消去すると保存した問題も削除されます
> - ブラウザの保存容量には上限があります。大量のファイルを保存すると容量が不足する場合があります

## 6. 一括採点（複数生徒の同時採点）

### 一括採点の流れ

1. メイン画面で「一括採点」タブを選択
2. 「生徒を追加」ボタンで生徒カードを追加（**最大10名**まで）
3. 各生徒カードに名前を入力し、答案ファイルをアップロード（**1生徒あたり最大5ファイル**）
4. 共通の模範解答をアップロード
5. 問題番号と採点基準を設定
6. 「一括採点開始」をクリック

> ⚠️ **ご注意**: 一括採点ではN名分の採点で**採点回数をN回**消費します。開始前に確認ダイアログが表示されるので、残り回数をご確認ください。

### 生徒カードの使い方

| 操作 | 方法 |
|------|------|
| **生徒の追加** | 「生徒を追加」ボタンをクリック |
| **名前の設定** | 各カードの名前欄に生徒名を入力 |
| **答案のアップロード** | 各カードのアップロード欄にファイルをドラッグ＆ドロップ |
| **生徒の削除** | 各カードの削除ボタンをクリック |

> 💡 **ヒント**: 生徒名はPDFレポートに記載されるため、正しい名前を入力してください。

### 進捗確認

- 処理中は進捗バーで状況を確認できます
- 各生徒の採点完了後、順次結果が表示されます

### 一部失敗した場合

一括採点で一部の生徒だけ失敗した場合は、「失敗した生徒を再試行」ボタンで、失敗した生徒分だけ再採点できます。成功した生徒の結果はそのまま保持されます。

### 結果のダウンロード

一括採点が完了すると、全生徒のPDFレポートをまとめたZIPファイルをダウンロードできます。

## 8. アカウント管理

### ログイン情報

- メールアドレスとパスワードでログイン
- パスワードを忘れた場合は「パスワードをお忘れですか？」からリセット
- メールリンクでのログイン（パスワード不要）にも対応

### デバイス制限について

**1アカウントにつき2台まで**のデバイスで利用可能です。

3台目のデバイスでログインしようとすると：
1. 「デバイス上限に達しました」というメッセージが表示
2. 既存デバイスの一覧が表示
3. 不要なデバイスを選択して削除
4. 新しいデバイスでログイン可能に

> 💡 **なぜ制限があるの？**: 教育機関でのアカウント不正共有を防ぎ、公正なサービス提供を維持するためです。1アカウントは個人での利用を想定しています。

### 使用状況の確認

「使用状況」ページで以下を確認できます：
- 今月の採点回数
- 残り利用可能回数
- プラン更新日

## 9. 推奨環境・ファイル形式

### 対応ブラウザ

| ブラウザ | 対応状況 |
|----------|----------|
| Chrome | ✅ 推奨 |
| Safari | ✅ 対応 |
| Firefox | ✅ 対応 |
| Edge | ✅ 対応 |

### 対応ファイル形式

| 形式 | 拡張子 | 備考 |
|------|--------|------|
| JPEG | .jpg, .jpeg | 推奨 |
| PNG | .png | 推奨 |
| PDF | .pdf | 複数ページ対応 |
| HEIC | .heic | iPhone写真に対応 |
| HEIF | .heif | 対応 |
| GIF | .gif | 対応 |
| WebP | .webp | 対応 |
| BMP | .bmp | 対応 |

### ファイルサイズ

- **最大**: 50MB/ファイル（画像は自動圧縮されます）
- **推奨**: 2MB以下（処理速度向上のため）
- **最大ファイル数**: 1回の採点につき10ファイルまで

> 💡 **画像の自動圧縮について**: アップロードされた画像は自動的に最適化されます。文字認識に必要な品質は保たれるよう調整されているため、スマホで撮影した高画質の写真でもそのままアップロードしてください。文字の多い答案では高品質モード、それ以外では通常モードが自動で適用されます。圧縮中は進捗バーが表示されます。

### 撮影のコツ

📸 **きれいに撮影するポイント**：

1. **明るい場所で撮影** - 自然光または明るい照明の下で
2. **真上から撮影** - 斜めからだと文字が歪む
3. **影を避ける** - 手や機器の影が入らないように
4. **全体を収める** - 答案全体がフレーム内に入るように
5. **ピントを合わせる** - 文字がくっきり見えるまで待つ
6. **枚数に注意** - 1回の採点では5枚以下が推奨（6枚以上で読み取り精度が低下する場合があります）

> 💡 **スキャナーがある場合**: スキャン画像の方が読み取り精度が高くなります（推奨解像度: 300dpi）

## 10. 困ったときは

### よくあるトラブルと対処法

| 症状 | 対処法 |
|------|--------|
| 手書き文字が正しく読み取れない | 明るい場所で真上から再撮影するか、読み取り結果を手動修正して再採点 |
| ファイルサイズが大きすぎると表示される | 画像は自動圧縮されますが、PDFの場合は枚数を減らすかオンライン圧縮ツールをご利用ください |
| 「混雑しています」と表示される | しばらく時間をおいてから再度お試しください |
| セッションが切れたと表示される | 再度ログインしてください |

- [よくある質問（FAQ）](./FAQ.md) をご確認ください
- それでも解決しない場合は、サポートまでお問い合わせください
