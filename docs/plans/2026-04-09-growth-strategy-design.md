# Taskal AI 成長戦略設計書

## 作成日: 2026-04-09

## 現状診断

### 状況
- 流入がほぼゼロ（無料トライアル利用者もほぼなし）
- SNS投稿（数回）・知人紹介を試みたが反応なし
- ターゲット: 学習塾の先生・個人家庭教師

### 根本原因（3層構造）

```
【認知層】 誰も存在を知らない
   └ SNS数回・知人紹介だけでは塾講師に届いていない

【信頼層】 知っても買えない
   └ 運営者不明・特商法404・実績ゼロ・精度不明

【定着層】 試しても続かない
   └ 採点履歴なし・生徒管理なし＝使い捨てで終わる
```

---

## 戦略1: 緊急の信頼修復

### 即日対応
| 施策 | 理由 |
|---|---|
| `/tokushoho`（特定商取引法）ページを修正 | 法的義務。現在404で致命傷 |
| 運営者の実名・顔写真・塾歴をLP上部に追加 | 教育者は「誰が作ったか」を必ず確認する |
| 使用AIモデルの明示（OpenAI GPT-4o等） | 生徒データを処理するAIが不透明では不信感 |

### 今月中
- 採点精度を20問で自社検証し数字をLP掲載（「プロ講師との一致率82%」等）
- ベータ協力者10名を招待（3ヶ月無料）→ 実名テスティモニアル獲得

### LP構成の変更
```
【現状】                          【改善後】
1. ヒーロー（キャッチ）            1. ヒーロー（塾講師専用に絞る）
2. 採点デモ                        2. お悩み共感セクション ← 移動
3. 機能説明                        3. 採点デモ（ビフォーアフター）
4. お悩みセクション ← 遅すぎ       4. Social Proof ← 新設
5. Before/After比較                5. 機能説明（3つの特長）
6. 3つの特長                       6. Before/After比較
7. 使い方3ステップ                 7. 使い方3ステップ
8. 登録3ステップ（重複）← 削除    8. 料金プラン
9. 料金                            9. FAQ
10. FAQ                            10. CTA（最終）
```

### コピー改善
**現状**: 「どんな国語の答案も3分でプロ視点の添削が完了」
**改善案**: 「疲れていても、22時でも。採点基準はブレない。」
- サブ: 「だいたいこんな感じ」の採点を、論理的な根拠に変える。

### ターゲット絞り込み
- 「学習塾、学校、家庭教師、保護者様など」→ 「学習塾・家庭教師の先生向け」に限定
- 料金プランの先頭を「ライト（家庭向け）」から「塾向け」に変更

### CTA改善
- 「無料で5回試す」→「今すぐ1問添削してみる（30秒・登録不要）」

## 戦略4: PMF（製品改善）

### 追加すべき機能（優先度順）
1. **生徒管理＋採点履歴ダッシュボード** — 使うほど価値が増える仕組み
2. **バッチ採点モード** — 5人分まとめて処理（現場ワークフロー適合）
3. **保護者向け共有レポート自動生成** — 塾長が「塾全体で導入」を決断するトリガー
4. **設問テンプレート登録** — スイッチングコスト創出
5. **塾アカウント（サブアカウント管理）** — 法人課金への転換

### 価格改定の提案
```
現状（回数ベース）              改善案（生徒数ベース）
¥980/月 → 10回              ¥1,980/月 → 生徒10名まで
¥1,980/月 → 30回            ¥4,980/月 → 生徒30名まで
¥4,980/月 → 無制限           ¥9,800/月 → 無制限（塾全体）
```

### キラー機能仮説
**「採点レポートの保護者LINE配信」** — 塾が保護者に「うちはAI添削を導入している」と差別化できる
